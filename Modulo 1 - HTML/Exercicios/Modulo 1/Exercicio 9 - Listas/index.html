Lista de Definições

Inteligência Artificial (IA/AI)
Sistemas computacionais capazes de simular a inteligência humana, aprendendo e tomando decisões.
Machine Learning (Aprendizado de Máquina)
Subcampo da IA que permite aos computadores aprenderem sozinhos com base em dados, sem serem explicitamente programados para cada ação
Deep Learning (Aprendizado Profundo)
Um tipo avançado de machine learning que utiliza redes neurais artificiais complexas para imitar a forma como o cérebro humano processa informações
Big Data:
Refere-se a conjuntos de dados extremamente grandes, complexos e de rápido crescimento, que exigem tecnologias especiais para processamento.
LLMs (Large Language Models):
Modelos de linguagem de grande escala, como o GPT, treinados com grandes volumes de texto para gerar respostas humanas.
