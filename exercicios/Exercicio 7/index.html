Negrito / Destaque

Nessa frase podemos ver que AQUI É NEGRITO fiz aqui com a tecla "B" que significa bold.

Nesta frase por exemplo , tenho um AQUI É NEGRITO usando a tag "STRONG"

então qual a diferença para "STRONG" e "B"

STRONG:Usamos esta tag no sentido semântico, de sentido, quando uma palavra ou frase é strong\forte.

B:Usamos esta tag no sentido de forma apenas.

Itálico/Ênfase

Nessa frase podemos ver que AQUI É ITÁLICO fiz aqui com a tecla "I" que significa italic.

Nesta frase por exemplo , tenho um AQUI É ITÁLICO usando a tag "EM"

então qual a diferença para "I" e "EM"

EM: Usamos esta tag no sentido semântico, de sentido, quando uma palavra ou frase precisa de ênfase.

I: Usamos esta tag no sentido de forma apenas.
